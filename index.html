Core Capabilities

Advanced AI Modes

Powered by Google GenAI SDK tailored for coding logic.

  • Quick Mode: Gemini 2.5 Flash for instant edits.
  • Deep Think: Gemini 3 Pro with architectural reasoning budgets.
  • Web Grounding: Real-time documentation fetching via Google Search.
  • Semantic Search: Query your active codebase instantly.

Powerful Code Canvas

A robust editor environment built for modern development.

  • Multi-Tab Editor: Syntax highlighting for TS, Py, HTML, CSS.
  • Live Preview: Instant interactive preview with responsive testing tools.
  • Bug Repair Panel: Dedicated error analysis workspace.
  • Smart Context: Right-click AI actions (Explain, Test, Refactor).

Developer Experience

Seamlessly integrated into your existing workflow.

  • GitHub Import: Load public repos directly.
  • Local Management: Browser storage project states.
  • Visual Diffing: AI changes presented as clear diffs.
  • Floating Toolbar: Context-aware tools at your cursor.
